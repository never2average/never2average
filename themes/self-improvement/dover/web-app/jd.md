# Full-Stack Intern (UI Integrator), Dover Web App

Security today is theater: rule libraries, dashboards of noise, and CVEs published long after the door was already open. [Dover](../overview.md) is the opposite. It sandbox-clones your application, finds what is genuinely exploitable, and opens the pull request that fixes it. The web app is where a team comes to see that and believe it.

I am looking for a builder more than a credential. If you can ship a full-stack surface fast and make a hard idea feel obvious, you can learn the security as you go.

## What you will do
- Build the web surface that turns Dover's findings and auto-PRs into something a team will act on.
- Wire the UI end to end with the harness backend.
- Make the trust path unmistakable: what was found, why it is exploitable, what the fix does.

## You are
- A strong full-stack builder who owns a UI from API to pixels.
- Curious about security; formal appsec background is a bonus, not a requirement.

## The deal

Paid stipend. Part-time and flexible around your life. Remote, global. If you are exceptional, this converts to a full-time or founding role.
