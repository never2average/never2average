# Approach

Mynt is the v2 financial-product layer: the version where plain-language financial intent becomes a more durable product surface. Agent Bank proves that an agent can assemble first-pass financial logic from a user's desired outcome. Mynt should turn that into a repeatable system for creating, reviewing, and operating financial products.

The product should connect intent to execution through explicit stages:

- Intent: what the user wants the money or financial workflow to accomplish.
- Logic: the inferred product rules, assumptions, constraints, and edge cases.
- Review: the human-visible explanation of risk, tradeoffs, and regulatory sensitivity.
- Operation: the monitored product state after the logic becomes live.

Mynt should feel chat-first, but it cannot be chat-only. The end state has to be financial product logic that can be inspected, tested, revised, and governed.
